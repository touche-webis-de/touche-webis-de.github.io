Introduction

Let's look at the following argument:

Boxing leads to serious injuries.

Source: Sweating fighter is punched in the face - gettyimages

Chances are, you glanced at the photo before reading the entire argument. The image almost lets us feel the pain of the blow and hear the impact, making the point even more compelling. This demonstrates the power of visuals—they engage our senses and emotions, making them far more memorable than words alone.

Data

The latest version of the dataset is on Zenodo. The arguments.xml contains the claims for which images should be found. The dataset also contains the image collection for retrieval. Contact us for access to the image generation API. The current version of the dataset contains about 10% of the claims and images of the final dataset.

Evaluation

Determining whether an image aligns with an argument is often subjective. For a strong evaluation, arguments are chosen to allow clear and effective visualization. In addition, key aspects are defined for each argument, which specify what the corresponding image should represent.
